# Booking System Frontend

Frontend part of booking system.

## Tech stack

- Next.js (App Router)
- TypeScript
- React Query
- Axios

## Features

- Users management
- Modal-based UI
- Booking creation
- Booking editing & cancellation
- Client booking list

## Architecture

- API layer (`lib/api.ts`)
- Business logic via hooks
- UI components

## Run locally

```bash
npm install
npm run dev
```
